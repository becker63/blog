# Curator Task

Select one strong writing seed from the provided work events, taste profile, previous insights, and repo evidence.

The output should be an insight card with:

- why this was selected
- what changed or was inspected
- a hidden thesis
- concrete evidence from commits, files, docs, or tests
- possible blog hooks
- relation to previous writing
- follow-up questions

If the evidence does not support a meaningful thesis, return `no_insight`.
